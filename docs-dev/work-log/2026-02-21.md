## 2026-02-21 array_int_element の bisect/決定変数制御の改善

**実施内容**
- `src/fzn/model.cpp` の `array_int_element` パース処理で、単調配列と非単調配列で挙動を分岐するよう変更
  - `set_no_bisect(index_var)`: 非単調配列の場合のみ呼び出し（単調配列では bisect を有効のまま維持）
  - `set_defined_var(result_var)`: 単調配列の場合のみ呼び出し（非単調配列では result を探索対象として残す）

**設計判断・理由**
- 単調配列用の `IntElementMonotonicConstraint` は bounds propagation が O(log n) で効くため、bisect の方が enumerate より効率的
- 非単調配列では index が決まっても result の propagation が不完全になりうるため、決定変数から外さない方が安全

- `src/fzn/model.cpp` (line 723-733)
