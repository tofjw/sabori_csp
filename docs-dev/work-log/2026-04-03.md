## 2026-04-03 10:17 bool_clause bump_activity 実験

**実施内容**
- `BoolClauseConstraint::bump_activity` のオーバーライドを複数パターンで実装・ベンチマーク評価
  - パターン1: trigger変数 + 2WLのもう一方に `1/n` ずつ
  - パターン2: 同上を `0.5/n` に変更
  - パターン3: trigger + 2WL に `0.4/n`、残りの変数に `0.2/n` を案分

  - パターン5: `n <= 10` の場合に全変数に `1/(2^(n-1))` 加算

**設計判断・理由**
- bool_clause は SAT節であり、2WL の構造を活かした activity bump を試みたが、効果なし
- 短い節（n <= 10）に絞る指数減衰やlog減衰も改善に繋がらなかった
