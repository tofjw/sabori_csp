## 2026-02-19 int_lin_eq defined_var 選択の改善

**実施内容**
- `int_lin_eq` の defined_var ヒューリスティック選択を改善
  - 最初に見つかった適格変数を即選択 → 全候補を走査し domain size 最大の変数を選択するように変更
  - 領域が大きい変数を探索候補から外すことで探索空間の削減効果を向上

**設計判断・理由**
- `Domain::size()` は inline かつ O(1) なので、全候補走査のコストは元のループと同等
- 領域が大きい変数を defined_var にすることで、その変数の値が他変数から決定され、分岐対象から除外される
