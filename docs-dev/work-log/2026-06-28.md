## 2026-06-28（続）probe 結論を vs CP-SAT で訂正——指標の乖離

**実施内容**
- 指摘: 上記 probe の結論（−23＝有害・既定オフ候補）は **probe_on vs probe_off の直接対決（目的値）**で出したもの。実際の Challenge のゴールは **CP-SAT（フィールド）に勝つこと**で指標が違う。`SABORI_PROBE=1/0 python bench_compare.py 2016 2025`（Sabori vs CP-SAT 勝敗）で比較。
- 結果（2016+2025 合算、Sabori 勝 / CP-SAT 勝 / 引分）:
  - **probe ON（既定）: 15 / 17 / 6**
  - **probe OFF: 12 / 17 / 9**
  - → **両年とも probe ON の方が Sabori 勝が多い（15 vs 12、CP-SAT 勝は同数）**。probe は引分を Sabori 勝に変え、CP-SAT には余計に負けていない。
- 「既定オフ候補」を撤回。§7 を「直接対決では負け（−23）だが対 CP-SAT では勝ちを増やす＝残すのが正解」に訂正（JP/EN）。**方法論の注記**を追加。
- `bench_probe.py` の YEARS を `SABORI_BENCH_YEARS` env で指定可能に（既定 2023,2024）。

**設計判断・理由**
- **直接対決（平均目的値）と vs フィールド（CP-SAT に勝つ）は別指標**。probe は「多くの問題で目的値を僅かに悪化させるが、数問で CP-SAT を超える閾値を押し上げて勝ちに変える」ため、直接対決マイナス・対フィールドプラスに乖離した。

- 教訓: ablation の指標選択が結論を左右する。負け/有害系の結論は、本来のゴール指標（vs フィールド）で裏取りすべき場合がある。

## 2026-06-28 21:48 証拠の等級分け（荷重を担う net と方向性 net の分離）
